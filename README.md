# ritik_patel_exercise_2

A Flutter memory matching game.

## Project Overview

Memory Match Game is a Flutter/Dart mobile game application designed for a simple memory matching challenge. The app allows users to play a card matching game where all cards are hidden at the beginning. The user selects two cards at a time and tries to find matching pairs.

If the two selected cards are the same, they remain open. If the cards are different, they flip back after a short delay. The game continues until all matching pairs are found. After completing the game, a congratulations message is displayed showing the total number of moves used by the player.

The main goal of this project is to demonstrate Flutter mobile app development, Dart programming logic, state management, user interaction, and game flow implementation using Android Studio.
